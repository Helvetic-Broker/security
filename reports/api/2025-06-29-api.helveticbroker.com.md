- **Risk:** Low
- **Instances:** 23
- **Description:**  
  The HTTP Strict-Transport-Security (HSTS) header is not set on some responses. This header helps protect against protocol downgrade attacks and cookie hijacking.
- **Recommendation:**  
  Ensure that your server sets the HSTS header on all HTTPS responses.
